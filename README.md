# SpringBootJsonDB

## Project Description
SpringBootJsonDB is a micro project developed using Spring Boot and JsonPowerDB.  
The project demonstrates CRUD (Create, Read, Update, Delete) operations using REST APIs.

This project was created using concepts learned from the JsonPowerDB course.

---

## Technologies Used
- Java
- Spring Boot
- JsonPowerDB
- REST APIs
- Maven
- Postman

---

## Features
- Add Student Record
- Fetch Student Record
- Update Student Record
- Delete Student Record

---

## Project Structure
- Controller Layer
- Service Layer
- Model Layer

---

## API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| POST | /save | Save Student Data |
| GET | /get/{id} | Get Student Data |
| PUT | /update | Update Student Data |
| DELETE | /delete/{id} | Delete Student Data |

---

## API Testing
All APIs were tested using Postman.

---

## Benefits of JsonPowerDB
- Lightweight and fast
- Simple API integration
- JSON-based database
- Easy CRUD operations
- Beginner friendly

---

## How to Run the Project

1. Clone the repository
2. Open the project in IntelliJ IDEA or Eclipse
3. Configure JsonPowerDB credentials
4. Run the Spring Boot application
5. Test APIs using Postman
